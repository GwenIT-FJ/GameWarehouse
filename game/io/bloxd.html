Bloxd.io

Bloxd.io is an exciting sandbox game with multiple game modes. In survival mode, you'll need to gather resources, craft tools, and defend against enemies to stay alive. The game features challenging obstacle courses that require precise jumping and movement skills.

Game Controls

  • • WASD - Move
  • • Shift or double-tap W - Run
  • • C, Z, \ or Caps Lock - Crouch
  • • T or Enter - Chat
  • • B - Open Shop
  • • / - Start Command
  • • O - Open Menu
