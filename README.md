# Game Warehouse

A modern, responsive game collection website featuring various HTML5 games.

## Project Overview

Game Warehouse is a curated collection of entertaining HTML5 games that can be played directly in the browser. The website features a clean, Apple-inspired design and is fully responsive for both desktop and mobile devices.

## Features

- Responsive design for all devices
- Apple-inspired color scheme
- SEO optimized structure
- Direct game embedding from HTML5Games.com
- Clean and intuitive user interface

## Technical Stack

- Next.js 14
- Tailwind CSS
- HTML5
- Responsive Design

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Project Structure

```
/
```
